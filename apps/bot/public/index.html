Omega

Self-Coding AI Discord Bot 18+ Self-Built Tools

The Concept

Omega is a Discord bot that writes its own code. Started with zero functionality, it evolved through conversations to build 18+ tools, and continues to grow based on user needs.

When you ask for a feature, Omega writes the code, commits the changes to git, deploys the update, and uses the new feature. You're not installing a pre-built bot. You're growing an AI that codes itself.

Special Capabilities

Self-Modifying Codebase: Omega can edit its own source code and deploy updates automatically.

Daily Automation: Generates philosophical blog posts at 9 AM UTC combining Hacker News insights with market predictions.

Error Monitoring: Automatically detects Railway runtime errors and creates GitHub issues with AI-powered analysis.

Multi-Language Execution: Safely executes code in 11 languages using Unsandbox API with isolated containers.

Technology Stack

Runtime: Node.js 18+ with TypeScript 5.x
AI: OpenAI GPT-4.1-mini via Vercel AI SDK v6
Framework: Discord Interactions API (serverless)
Build: Turborepo + pnpm 9
Database: SQLite
Hosting: Railway (production)
