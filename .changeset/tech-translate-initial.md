Initial release of @omega/tech-translate - AI-powered technical specification generator

Features:
- Convert user requests into comprehensive technical specifications
- Support for multiple output formats (Markdown, JSON, both)
- Pluggable LLM providers (OpenAI, Anthropic)
- CLI tool with rich options
- TypeScript-first with strict types
- ESM + CJS builds for maximum compatibility
- Comprehensive documentation and examples
- Rate limiting and retry logic built-in
- PAM-compliant output schema
